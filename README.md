# Utility functions for MATLAB

A set of utility functions to efficiently organize, filter, and process `table` data in MATLAB.

For a detailed explanation and background of these tools (in Japanese), please refer to this [Qiita article](https://qiita.com/km_MATLAB_Amb/private/fe787b7e2ad38d59239f).

## List of Functions

- `filterTable`: Filters rows by specified variable–value pairs.
- `splitTableByAxes`: Splits a table into a multidimensional cell array based on all combinations of specified axis variables.
- `reduceTableByAxes`: Splits a table based on all combinations of specified axis variables and then applies a reduction function to a data variable.

## Syntax

### `filterTable`
```matlab
[Tfilt, idx] = filterTable(T, filterVar1, filterValue1, filterVar2, filterValue2, ...)
```
Extracts rows from table `T` that match all the specified values. You can specify any number of pairs of variable names (`filterVar`) and their corresponding values (`filterValue`).

**Inputs:**
- `T` (table): Table to be filtered.
- `filterVar` (string): Variable name(s) used for filtering.
- `filterValue`: Value(s) to filter by.

**Outputs:**
- `Tfilt` (table): Table containing only the rows that match the specified values.
- `idx` (logical vector): Logical index of the rows that matched the specified values.

### `splitTableByAxes`
```matlab
[Tsplit, axisValues] = splitTableByAxes(T, axisVars)
```
Splits the table `T` based on combinations of the variables specified in `axisVars`.

**Inputs:**
- `T` (table): Table to be split.
- `axisVars` (string vector): List of variable names to use as axes.

**Outputs:**
- `Tsplit` (cell array of tables): Multidimensional cell array containing the split tables. Each dimension corresponds to each variable in `axisVars`.
- `axisValues` (struct): Structure containing the unique values of each axis variable.

### `reduceTableByAxes`
```matlab
[reducedValues, axisValues] = reduceTableByAxes(T, axisVars, reduceFcn, dataVar)
```
Splits the table `T` by `axisVars` and applies a reduction function `reduceFcn` to the variable `dataVar` for each group.

**Inputs:**
- `T` (table): Table to be reduced.
- `axisVars` (string vector): List of variable names to use as axes.
- `reduceFcn` (function_handle): Reduction function to apply.
- `dataVar` (string): Variable name containing the values to be reduced.

**Outputs:**
- `reducedValues` (multidimensional array): Array containing the reduced results.
- `axisValues` (struct): Structure containing the unique values of each axis variable.

## Usage Examples

First, let's create a sample table `T` containing 
variables `X`, `Y`, `Trial`, and `Z`, where `Z` is generated by `Z = X .* Y + 0.1 * randn`:
```matlab
rng(1)
[X, Y, Trial] = ndgrid(1:2, 1:2, 1:3);
Z = X .* Y + 0.1 * randn(size(X));
T = table(X(:), Y(:), Trial(:), Z(:), 'VariableNames', ["X", "Y", "Trial", "Z"]);
```

```
T =

  12×4 table

    X    Y    Trial       Z   
    _    _    _____    _______

    1    1      1       0.9351
    2    1      1       2.1181
    1    2      1       1.9242
    2    2      1        3.889
    1    1      2      0.91544
    2    1      2       1.9427
    1    2      2       1.9441
    2    2      2       4.0178
    1    1      3      0.98031
    2    1      3       2.0586
    1    2      3       1.9148
    2    2      3         4.08
```

### 1. Extracting rows with `filterTable`
Extract rows where `X` is 1 and `Y` is 2:

```matlab
[Tfilt, idx] = filterTable(T, "X", 1, "Y", 2)
```
**Output:**
```text
Tfilt =

  3×4 table

    X    Y    Trial      Z   
    _    _    _____    ______

    1    2      1      1.9242
    1    2      2      1.9441
    1    2      3      1.9148
```

### 2. Splitting a table with `splitTableByAxes`
Split table `T` by combinations of `X` and `Y`:

```matlab
[Tsplit, axisValues] = splitTableByAxes(T, ["X", "Y"])
```
**Output:**
```text
Tsplit =

  2×2 の cell 配列

    {3×4 table}    {3×4 table}
    {3×4 table}    {3×4 table}


axisValues = 

  フィールドをもつ struct:

    X: [2×1 double]
    Y: [2×1 double]
```
`Tsplit{1, 2}` is a 3x4 table containing the rows where `X==1` and `Y==2`.

### 3. Aggregating values with `reduceTableByAxes`
Calculate the mean of `Z` across trials for each combination of `X` and `Y`:

```matlab
[meanZ, axes] = reduceTableByAxes(T, ["X", "Y"], @mean, "Z")
```
**Output:**
```text
meanZ =

    0.9436    1.9277
    2.0398    3.9956


axes = 

  フィールドをもつ struct:

    X: [2×1 double]
    Y: [2×1 double]
```
In this case, `meanZ` is a 2x2 matrix which approximates `axes.X .* axes.Y' = [1, 2; 2, 4]`.
